# UnitConverter.live Launch Checklist

## Technical
- [ ] All pages load without console errors
- [ ] Internal links return HTTP 200
- [ ] `robots.txt` reachable at `/robots.txt`
- [ ] `sitemap.xml` reachable at `/sitemap.xml`
- [ ] Canonical URL present on all pages
- [ ] `meta description` present and unique

## SEO
- [ ] Google Search Console property verified
- [ ] Sitemap submitted in Search Console
- [ ] Homepage and top tool pages requested for indexing
- [ ] OG tags verified for homepage share cards

## AdSense readiness
- [ ] Privacy Policy linked site-wide
- [ ] Terms of Service linked site-wide
- [ ] Contact method visible (`admin@unitconverter.live`)
- [ ] No misleading UI elements or broken pages
- [ ] Helpful original content present on each major tool page

## UX
- [ ] Mobile layout checked (small screens)
- [ ] Form reset/clear behavior checked
- [ ] Calendar print-to-PDF checked
- [ ] Typing/BMI/Mortgage edge-cases checked
